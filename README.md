# Sales Performance Dashboard

## Project Overview
This project analyzes sales data to track performance across regions, categories, and customer segments using SQL, Excel, and Power BI.

## Objectives
- Track key performance indicators (KPIs) such as total sales, profit, average order value, and discount impact
- Analyze performance across regions, categories, and customer segments
- Identify top-performing and low-performing areas
- Support business decision-making using data insights

## Key Insights
- Identified top-performing regions contributing significantly to overall revenue
- Analyzed the impact of discounts on profit and sales performance
- Highlighted underperforming segments and regions for improvement
- Provided insights to improve sales strategy and reporting efficiency

## Tools Used
- SQL (Data analysis)
- Excel (Data cleaning)
- Power BI (Dashboard creation)

## Process
- Imported raw sales data
- Cleaned and prepared data using Excel
- Performed data analysis using SQL queries
- Built data model using star schema in Power BI
- Created interactive dashboards with multiple pages for detailed insights
