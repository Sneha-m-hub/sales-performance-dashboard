# Sales Performance Dashboard

## Project Overview
This project presents a sales performance dashboard built using Power BI to analyze key business metrics and trends.

## Key Insights
- Total sales reached 56.7M with profit of 7.15M
- Technology category generated highest sales
- West region showed strong performance
- Top customers contributed major revenue
- Recent trend shows decline in sales

## Tools Used
- Excel
- Power BI
- SQL
